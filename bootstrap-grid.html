Bootstrap Grid System and Respective classes

Based on the view port size bootstrap will support max of 12 columns

by using row class we can create grid and define respective columns under that

There mainly we use three column classes

col-lg-* : lg stands for large, for large screens we can use this class

col-md-* : md stands for medium , for medium and large screens we can use this class

col-xs-* : xs stands for extra small, for small,medium and large we can use this class

along with this we have extra large as well
