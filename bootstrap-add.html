Bootstrap installation

To install bootstrap visit getbootstarp.com

From here either we can download bootstrap zip file with customization or we can use CDN

In this project I am using CDN, we have to place the css CDN under header tag and js cdn under body tag

To get cdn you can visit this site bootstrapcdn

Once you past the CDN or add the files to project we are done with bootstrap integration. If you are working in local add https to make it work else in live sever you don't need to add this.

Along with these add jQuery latest version to the project.

You can get jQuery cdn from here jQuery CDN
