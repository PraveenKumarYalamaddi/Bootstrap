Bootstrap buttons and respective classes

Bootstrap have many button classes based on the scenario we can use them

below are the popular button classes.

btn : it is a base class, this will button features

btn btn-default : it's a default button class with white color

btn btn-primary : primary button comes with blue color

btn btn-success : success button comes with green color

btn btn-info : md info button comes with light blue color

btn btn-warning : md warning colored button

btn btn-danger : danger colored button

btn btn-link : if you want to provide any link with button we can use this class
