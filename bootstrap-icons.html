Bootstrap icons & from controls

Bootstrap have many inbuilt icon classes for the purpose of frequent use, we don't need to maintain any resources and they will reduce your project size as well.

There are bunch of icons under glyphicon package, if you want to use any icon you must need to include base class glyphicon before that class

Some of the examples are here.

glyphicon : This is a base class and it must be included before all the icon classes for the performance reason.

glyphicon glyphicon-cloud

glyphicon glyphicon-user

glyphicon glyphicon-print

glyphicon glyphicon-apple

glyphicon glyphicon-thumbs-up

glyphicon glyphicon-thumbs-down
